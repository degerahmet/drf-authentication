Django Rest Framework Authentication (Knox & JWT)
=================================================

Abstract
--------

Django Rest Framework JSON Web Token & Knox Token authentication plugin for the `Django REST
Framework <http://www.django-rest-framework.org/>`_.

Features
=========


* User Registration
* Login/Logout
* Retrieve/Update the Django User model
* Password change
* Password reset via e-mail
